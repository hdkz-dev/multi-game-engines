# @multi-game-engines/adapter-ensemble

Next-generation ensemble (swarm) engine adapter for multi-game engines.

## Features

- **Swarm Intelligence**: Combines multiple engines to achieve higher accuracy.
- **Configurable Strategies**: Supports Majority Vote, Best Score, and Weighted consensus.
- **Protocol Agnostic**: Can bundle engines using different protocols (UCI, USI, etc.) under a single interface.

## Installation

```bash
pnpm add @multi-game-engines/adapter-ensemble
```
