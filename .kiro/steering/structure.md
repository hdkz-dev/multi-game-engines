# Project Structure

## Directories

- `packages/core`: Core logic, interfaces, and common utilities.
- `packages/adapter-*`: Specific engine implementations (Stockfish, Yaneuraou, etc.).
- `docs/`: Comprehensive project documentation (Architecture, Tasks, etc.).
- `scripts/`: Development and maintenance scripts.
- `.kiro/`: Spec-driven development settings and historical specifications.

## Key Files

- `GEMINI.md`: AI agent context and role definition.
- `TASKS.md`: Technical debt and pending tasks tracking.
- `DECISION_LOG.md`: Architecture Decision Records (ADR).
- `PROGRESS.md`: Current implementation status and test coverage.
